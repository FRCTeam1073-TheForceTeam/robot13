Member Enumeration Documentation

enum SmartGyro::GyroMode

Different options to represent how your angle will be returned when SmartGyro::GetAngle() is called.

Enumerator
radiansOverPi 

Angles are represented by radians divided by pi this is how the WPILib Gyro class returns angles

radians 

Angles are represented by radians

degrees 

Angles are represented by degrees

Constructor & Destructor Documentation

SmartGyro::SmartGyro ( UINT32  channel)

Constructs a SmartGyro at a specified port. By default, objects of SmartGyro will behave exactly like the WPI Gyro meaning that SmartGyro::gyroMode will equal radiansOverPi.

Parameters
channelThe Analog Channel port on your robot where the SmartGyro is plugged into.

float SmartGyro::GetAngle ( )

Returns the angle that the Gyro is set at. Overrides Gyro::GetAngle() to convert the returned float into Degrees, Radians, or Radians divided by pi depending on GyroMode.
